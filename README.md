# ⚡ YAMD - Ultra Speed Downloader

**YAAQOB ALMAHAJERI MEDIA DOWNLOADER | ULTRA SPEED EDITION**

بوت تلغرام متقدم لتحميل الفيديوهات والوسائط من جميع المنصات الشهيرة بأقصى سرعة ممكنة.

## 🚀 المميزات الرئيسية

- ✅ **تحميل فوري** من جميع المنصات (YouTube, TikTok, Instagram, Facebook, Pinterest وغيرها)
- ✅ **دعم Invidious** لتجاوز حجب YouTube
- ✅ **تقسيم ذكي** للملفات الكبيرة (أكثر من 48 MB) بتلقائياً
- ✅ **تحميل وسائط** بجودات متعددة (144p, 240p, 360p, 480p, Best Quality)
- ✅ **استخراج صوت** (MP3) من الفيديوهات
- ✅ **شريط تقدم ديناميكي** يعرض السرعة والوقت المتبقي
- ✅ **قاعدة بيانات** لتتبع المستخدمين والتحميلات
- ✅ **لوحة إدارية** متقدمة للمشرفين
- ✅ **يعمل على أضعف الشبكات** بفضل معالجة الأخطاء المتقدمة

## 📦 المتطلبات

- **Python 3.11+**
- **FFmpeg** (لتقسيم الملفات والمعالجة)
- **sqlite3** (مدمج في Python)

### المكتبات المطلوبة:

```bash
python-telegram-bot[httpx]>=20.0
yt-dlp
httpx
```

## 🔧 التثبيت والإعداد

### 1. استنساخ المستودع

```bash
git clone https://github.com/jacobkhaled01-spec/YAMD.git
cd YAMD
```

### 2. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 3. تثبيت FFmpeg

**على Linux:**
```bash
apt-get update && apt-get install -y ffmpeg
```

**على macOS:**
```bash
brew install ffmpeg
```

**على Windows:**
- قم بتحميل FFmpeg من [ffmpeg.org](https://ffmpeg.org/download.html)
- أضفه إلى متغيرات البيئة (PATH)

### 4. إعداد متغيرات البيئة

قم بإنشاء ملف `.env` أو تعيين متغيرات البيئة:

```bash
export BOT_TOKEN="your_bot_token_here"
export ADMIN_ID="your_admin_id_here"
```

الحصول على:
- **BOT_TOKEN**: من [@BotFather](https://t.me/botfather) على Telegram
- **ADMIN_ID**: معرف حسابك على Telegram

### 5. تشغيل البوت

```bash
python bot.py
```

## 🐳 التشغيل باستخدام Docker

### بناء الصورة:

```bash
docker build -t yamd-bot .
```

### تشغيل الحاوية:

```bash
docker run -e BOT_TOKEN="your_token" -e ADMIN_ID="your_id" yamd-bot
```

## 📝 طريقة الاستخدام

### للمستخدمين العاديين:

1. **ابدأ البوت:**
   - أرسل `/start` للحصول على معلومات الترحيب

2. **حمل فيديو:**
   - ببساطة أرسل رابط الفيديو من أي منصة
   - سيظهر شريط تقدم يعرض السرعة والحجم والوقت المتبقي

3. **اختر الجودة:**
   - بعد أول محاولة، يمكنك اختيار جودة محددة من الأزرار

4. **المزيد من المعلومات:**
   - استخدم `/about` للحصول على معلومات إضافية

### للمشرفين:

استخدم `/admin` للوصول إلى لوحة التحكم:

- 👥 **المستخدمون**: عرض آخر 15 مستخدم نشطين
- 📜 **آخر التحميلات**: عرض آخر 20 تحميلة مع التفاصيل
- 📊 **الإحصائيات**: عرض إجمالي المستخدمين والتحميلات والسرعة الوسيطة
- 🗑 **تنظيف مؤقت**: حذف جميع الملفات المؤقتة

## 🔐 قاعدة البيانات

البوت يستخدم SQLite لتتبع:

### جدول المستخدمين (users):
```sql
uid          - معرف المستخدم
username     - اسم المستخدم
fname        - الاسم الأول
seen         - وقت أول رؤية
active       - آخر نشاط
cnt          - عدد التحميلات
```

### جدول التحميلات (downloads):
```sql
id           - معرف التحميلة
uid          - معرف المستخدم
url          - رابط المصدر
title        - عنوان الفيديو
quality      - الجودة المختارة
size         - حجم الملف
speed        - سرعة التحميل (KB/s)
ts           - الطابع الزمني
```

## ⚡ الخصائص التقنية

### جودات التحميل المدعومة:

| الرمز | الجودة | التنسيق |
|------|--------|---------|
| `144` | 🐢 144p | أقل حجم |
| `240` | 🐢 240p | منخفضة |
| `360` | 🎥 360p | متوسطة |
| `480` | 🎥 480p | عالية |
| `best` | ✨ أفضل جودة | أعلى جودة متاحة |
| `audio` | 🔊 MP3 | صوت فقط |

### حدود النظام:

- **أقصى حجم للملف:** 48 MB (يتم تقسيمه تلقائياً)
- **المجلد المؤقت:** `~/yamd_dl`
- **مهلة الاتصال:** 60 ثانية
- **عدد محاولات إعادة المحاولة:** 10
